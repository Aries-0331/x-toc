# XTOC v0.3.0 Promo Copy

## Twitter / X

### English

XTOC v0.3.0 is out.

You can now save excerpts from X/Twitter long-form articles:

- select text
- click `save to xtoc`
- manage excerpts in Options
- export Markdown or JSON for Obsidian, Notion, Logseq, or your own workflow

Built for readers who turn good threads into reusable notes.

### 中文

现在可以保存 X/Twitter 长文里的重点摘录：

- 选中文字
- 点击 `save to xtoc`
- 在 Options 管理摘录
- 导出 Markdown / JSON，同步到 Obsidian、Notion、Logseq 或自己的知识库

适合把长文阅读沉淀成笔记的人。

## Xiaohongshu

### English

I added excerpt saving to XTOC.

When reading a long article on X/Twitter, select any useful paragraph and save it locally. Later, open Options to review, select, delete, and export excerpts as Markdown or JSON.

The goal is not persistent highlighting. It is lightweight capture for personal knowledge workflows.

### 中文

给 XTOC 加了一个我自己很需要的新功能：保存长文摘录。

现在读 X/Twitter 长文时，可以直接选中一段内容，点击 `save to xtoc` 保存。之后在 Options 页面统一查看、勾选、删除，也可以导出 Markdown / JSON。

这个功能不是网页高亮工具，而是为了把阅读时看到的好内容沉淀到自己的知识库里，比如 Obsidian、Notion、Logseq。
